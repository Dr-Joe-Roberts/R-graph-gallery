Waffle chart




A waffle chart is a square alternative to the pie chart ! It represents the repartition of a numeric variable among several groups as a set of squares. This section displays many examples build with R and the waffle package.





Step by step with the waffle package

The waffle package is the best way to build a waffle plot with R. It comes with a waffle() function that will handle all the job for us. The first example below focuses on the data input format that is required. Following examples explain how to customize the chart and proceed basic needs like grouping with an additional level of hierarchy.
